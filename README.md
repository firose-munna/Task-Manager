# Task Manager

This is a flutter project with REST API.

## Features:
1. Sign up
2. Login
3. Update Profile
4. Add New Task
5. Task Status Change
6. Forgotten Password Recover
7. Delete Task
8. Four Types of Task Screen
9. Logout

## Upcoming Features:
1. Modify Task Details
2. Notification System
